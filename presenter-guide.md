### #3 Claude Code (1분 30초)

**키포인트**: 코딩만이 아니라 문서/분석/컴퓨터 조작까지 + 도구 비교 후 채택

> "저희가 사용하고 있는 도구는 Claude Code입니다. 코딩 에이전트로 알려져 있지만, 실제로는 코드 작성뿐 아니라 문서 작성, 데이터 분석, 컴퓨터를 직접 조작하는 것까지 가능합니다."

> "기존 ChatGPT 같은 챗봇과 다른 점은, 챗봇은 답변을 하지만 이건 직접 일을 합니다. 파일을 열고, 수정하고, 저장까지 스스로 합니다."

> (하단 설명 블록 가리키며) "Cursor 등 여러 도구를 비교 검토한 결과, Claude Code가 가장 우수하여 채택했습니다."

- **포인트**: "답변이 아니라 직접 일을 한다"

### #7 SW Dev. Portal (2분) ★핵심 슬라이드

**키포인트**: 10명·1년 vs 2명·4주 — 한 장에서 한눈에 비교

> "왼쪽이 2022년입니다. 프론트엔드 프레임워크를 전환하는 작업에 ITO 6명을 별도 계약하고 총 10여 명이 1년간 진행했습니다."

> "오른쪽이 올해입니다. 백엔드 프레임워크 전면 업그레이드인데, 파일 1,000개 이상, 10만 라인 수정, 라이브러리 의존성 전체 교체까지 — 2022년보다 오히려 더 복잡한 작업입니다."

- **3초 멈춤** — 청중이 숫자를 비교할 시간

> "단순히 빨리 끝내는 게 목적이 아니라, 시스템 전체를 AI 친화적인 구조로 전환하는 작업도 같이 하고 있습니다."

> "특히 AI를 활용해서 테스트 자동화 시스템을 구축했는데, 5,000개가 넘는 검증 시나리오에 커버리지 90% 이상을 확보했습니다. AI가 코드를 작성하면 자동으로 검증이 돌아가고, 최종 판단은 사람이 합니다."

> "AI는 도구이고, 책임은 엔지니어에게 있습니다."

## 예상 질문 & 답변 가이드

**Q: "컨트롤러 쪽 업무에도 활용 가능한가요?"**
> "현재 코드 기반 업무에서 효과가 가장 크고, FW나 설계 영역은 아직 적용 사례를 만들어가는 단계입니다. 다만 문서 작성이나 데이터 분석 같은 부분은 직군 상관없이 활용 가능성이 있습니다."

**Q: "보안 이슈는 없나요?"**
> "AI PoC 전용 외부망을 통해 진행하고 있고, SE에서 코드 보안 분류, 반출 프로세스, 형상 권한 관리를 별도로 수행하고 있습니다."

**Q: "Claude Code 비용은?"**
> "현재 전사 차원에서 AI센터를 통해 라이선스를 지원하고 있습니다."

**Q: "AI가 만든 코드 품질이 괜찮나요?"**
> "자체 검증 시스템으로 자동 테스트하고, 사람이 반드시 리뷰합니다. AI가 초안을 만들고 사람이 판단하는 구조입니다."

**Q: "ChatGPT랑 뭐가 다른가요?"**
> "ChatGPT는 대화형 답변을 생성하는 챗봇이고, Claude Code는 파일을 직접 읽고, 수정하고, 명령어를 실행하는 에이전트입니다."
