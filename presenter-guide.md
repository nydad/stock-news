> "기존 ChatGPT 같은 챗봇과 다른 점은, 챗봇은 '답변'을 하지만 이건 '직접 일'을 합니다. Cursor 등 다양한 도구를 비교 검토한 결과, 성능이 가장 우수하여 Claude Code를 채택했습니다."

- 3개 카드를 하나씩 짚으며 설명
- **포인트**: "답변이 아니라 직접 일을 한다"가 핵심 한 줄

## 예상 질문 & 답변 가이드

**Q: "컨트롤러 쪽 업무에도 활용 가능한가요?"**
> "현재 코드 기반 업무에서 효과가 가장 크고, FW나 설계 영역은 아직 적용 사례를 만들어가는 단계입니다. 다만 문서 작성이나 데이터 분석 같은 부분은 직군 상관없이 활용 가능성이 있습니다."

**Q: "보안 이슈는 없나요?"**
> "AI PoC 전용 외부망을 통해 진행하고 있고, SE에서 코드 보안 분류, 반출 프로세스, 형상 권한 관리를 별도로 수행하고 있습니다."

**Q: "Claude Code 비용은?"**
> "현재 전사 차원에서 AI센터를 통해 라이선스를 지원하고 있습니다."

**Q: "AI가 만든 코드 품질이 괜찮나요?"**
> "자체 TC 프레임워크로 자동 검증하고, 사람이 반드시 리뷰합니다. 100% 신뢰하는 게 아니라, AI가 초안을 만들고 사람이 판단하는 구조입니다. 이 검증 프로세스 자체도 BP 사례를 통해 계속 발전시키고 있습니다."

**Q: "ChatGPT랑 뭐가 다른가요?"**
> "ChatGPT는 대화형으로 답변을 생성하는 챗봇이고, Claude Code는 터미널에서 직접 파일을 읽고, 수정하고, 명령어를 실행하는 에이전트입니다. 사람이 지시하면 AI가 직접 일을 수행합니다."
