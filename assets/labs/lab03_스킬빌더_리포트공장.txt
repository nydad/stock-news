[Lab 2] Skill Builder — 리포트 공장 만들기
=============================================
소요: 35분  |  준비물: Lab 1에서 만든 리포트


핵심 개념:
  Lab 1에서 좋은 리포트를 "한 번" 만들었습니다.
  이제 이걸 Skill로 만들면, 주제만 바꿔서
  같은 품질의 리포트를 "무한히" 찍어낼 수 있습니다.
  → 결과물을 만드는 게 아니라, 공장을 만드는 것.


[Step 1] Skill Builder 설치 — Claude Code에 입력:
--------------------------------------------------

/install-skill https://github.com/anthropics/claude-code-skill-builder


[Step 2] 리포트 Skill 생성:
----------------------------

방금 만든 리서치 리포트의 구조와 스타일을 분석해서
같은 양식으로 아무 주제든 리포트를 만들 수 있는
Skill을 만들어줘.

Skill 이름: research-report
트리거: /research-report
기능: 주제를 입력하면 웹 검색 후 논문급 HTML 리포트 자동 생성


[Step 3] Skill 테스트 — 다른 주제로 돌려보기:
-----------------------------------------------

/research-report

주제: (Lab 1에서 안 한 다른 주제를 입력)

예시:
  "TSMC vs 삼성 파운드리 기술 격차 분석"
  "AI 에이전트 시장 2026 전망"
  "반도체 장비주 투자 분석"


[Step 4] Skill 개선 (여유가 있으면):
-------------------------------------

생성된 리포트가 마음에 안 드는 부분이 있으면:

"이 Skill에서 차트를 항상 3개 이상 포함하도록 수정해줘"
"executive summary를 항상 맨 위에 넣도록 수정해줘"
"출처를 각주로 정리하도록 수정해줘"

→ Skill 파일 위치: .claude/skills/research-report.md
→ 이 파일을 수정하면 앞으로 모든 리포트에 반영됩니다.


[참고] Skill 파일 구조:
------------------------

~/.claude/skills/이름.md          → 개인용 (모든 프로젝트)
프로젝트/.claude/skills/이름.md   → 프로젝트용 (해당 프로젝트만)

--- (YAML frontmatter)
name: research-report
description: 논문급 리서치 리포트 자동 생성
trigger: /research-report
---
(여기에 Claude에게 줄 지시사항)
